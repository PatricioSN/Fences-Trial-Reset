# Fences-Trial-Reset
A tool to study how Windows trial systems work (using Fences).

### How to use:

1. download the arquive and extract it to a folder or clone the repository.
2. Run the tool.
    1. The program will try to find and delete the license and cache files in the default path of the installation.
   2. If it fails, you will have two options:
        1. Manually inform the path of the files in the program.
        2. Run the auto mode of the program to find the files automatically.
3. Enjoy!
